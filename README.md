# Описание проекта:
Тестирование проекта [Codebattle](https://codebattle.hexlet.io/). 

Проект представляет собой игру по решению задач программирования на скорость. После регистрации пользователю доступен личный кабинет с доступом к статистике и настройкам. Функционал предусматривает возможность выбора сложности задач, противника, языка программирования для решения. Также пользователи могут общаться в чате, делиться приглашением на игру, следить за ходом игры других участников. 

Код проекта находится в открытом доступе, что дает возможность разработчикам принять участие в доработке сервиса.
